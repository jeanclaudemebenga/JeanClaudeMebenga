# Sahel Food Market Dynamics Streamlit Mini-App

## Files
- `app_sahel_market.py` — Streamlit app
- `requirements_streamlit_sahel.txt` — Python dependencies

## Run locally
```bash
pip install -r requirements_streamlit_sahel.txt
streamlit run app_sahel_market.py
```

## Data
By default, the app looks for a file named:
`wfp_food_prices_database_wacaro.csv`
in the same folder as the app.

You can also upload the CSV directly from the sidebar when the app opens.

## What the app includes
- Top staple commodities
- Hotspot market ranking
- Staple price trends over time
- Seasonal price patterns
- Imported rice vs local staples
- Market dispersion within countries
- Volatility heatmap
- Purchasing power chart

## Presentation tip
Use the sidebar filters live during your presentation to move from a regional Sahel story to a country-specific operational discussion.
